To execute this application,

Install
1) Python2.7
2) Flask Framework
3) Open CV2, Matplotlib and Numpy Libraries

Place this project folder as it is anywhere on your computer

Start the application by executing server.py file.

Open http://127.0.0.1:5000/ on your web browser.

***************************

All the folders inside static folder except SampleInput and sampleOutput folder,
are necessary for the proper execution of this application.
